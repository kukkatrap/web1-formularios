11:¿Qué son los códigos de estado de respuestas HTTP? Realizar una tabla con su clasificación. Bibliografía MDN

Los códigos de estado de respuesta HTTP indican si se ha completado satisfactoriamente una solicitud HTTP específica.

codigos de estado de respuestas HTTP
Enumeración Clasificación Ejemplo
100-199 Respuestas informativas Continue
200-299 Respuestas satisfactorias accepted
300-399 Redirecciones Multiple choise
400-499 Errores de los clientes Bad request
500-599 errores de los servidores Bad gateway
